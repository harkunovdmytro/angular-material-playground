Usual Mat Chips

Chips allow users to view information, make selections, filter content, and enter data.

MatChip

Material design styled Chip base component. Used inside the MatChipSet component.

Extended by MatChipOption and MatChipRow for different interaction patterns

Inputs

  • ariaDescription: string
  • ariaLabel: string
  • color: ThemePalette
  • disableRipple: boolean
  • disabled: boolean
  • highlighted: boolean
  • removable: boolean
  • id: string
  • role: string | null

Outputs

  • destroyed: EventEmitter MatChipEvent
  • removed: EventEmitter MatChipEvent

Others

  • basicChipAttrName: 'mat-basic-chip'
  • defaultColor: ThemePalette | undefined
  • defaultTabIndex: number
  • leadingIcon: MatChipAvatar
  • primaryAction: MatChipAction
  • removeIcon: MatChipRemove
  • trailIcon: MatChipTrailingIcon

    • MatChipOption extends MatChip

    An extension of the MatChip component that supports chip selection. Used with MatChipListbox.

    Unlike other chips, the user can focus on disabled chip options inside a MatChipListbox. The user cannot click disabled chips.

    Inputs

    • ariaDescription: string
    • ariaLabel: string
    • color: ThemePalette

    • disabled: boolean
    • highlighted: boolean
    • removable: boolean

    • selected: boolean
    • value: boolean
    • id: string
    • role: string | null

    Outputs

    • removed: EventEmitter MatChipEvent
    • selectionChange: EventEmitter MatChipSelectionChange
